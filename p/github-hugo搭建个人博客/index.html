github+hugo搭建个人博客

Jan 20, 2022
阅读时长: 2 分钟

由于hexo生成速度太慢，在这次整理博客时尝试使用hugo。记录搭建过程方便自己后面折腾的时候查看，也希望能帮助同样搭建博客的小伙伴节省时间。

参考文章

  • 浅谈我为什么从 HEXO 迁移到 HUGO - 少数派 (sspai.com)

  • Hugo中文文档 (gohugo.org)

  • 介绍 | Hugo 主题 Stack (jimmycai.com)

hugo 配置（windows）

git和go

hugo需要git和go环境

在git官网下载对应版本安装包进行安装

在go官网下载对应安装包

hugo

1.从hugo下载压缩包解压

2.为hugo目录设置环境变量

3.win+r打开cmd输入hugo version出现版本信息说明配置成功

完成上述之后hugo server和hugo new等命令均报错，未解决，遂改用wsl

hugo 配置（linux）

我使用win11的wsl（ubuntu20.04）

1.从hugo下载对应架构deb压缩包（用arch查看linux架构）

2.dpkg -i 压缩包 解压

3.hugo version 看是否安装成功

4.hugo server 在本地预览。默认端口为1313，也可以-p指定端口。在浏览器http://localhost:1313查看

主题安装与配置

安装

官网主题库Hugo Themes

1.git clone喜欢的主题或者下载zip解压到博客的themes文件夹中(优先选择有exampleSite的主题方便构建网站)

2.修改博客主目录下的配置文件将主题设置为将要使用的主题(要注意该名称与themes文件夹中主题文件夹名字一致)

stack主题配置

我选择了stack主题

发布文章

在content\post下使用markdown进行编写。md文档必须命名为index.md,否则会出现文章图片和封面图片无法显示的情况。 如要为文章设置封面，在开头标注image = “xxx.jpg”

设置分类

在content\categories下面进行分类修改
